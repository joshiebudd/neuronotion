How to add to Gmail

  1. Select everything inside a signature card below (from the logo to the pills)
  2. Copy it (Ctrl+C / Cmd+C)
  3. Go to Gmail > Settings (gear icon) > See all settings > General tab
  4. Scroll to the Signature section and click "+ Create new"
  5. Paste (Ctrl+V / Cmd+V) into the signature editor
  6. Set as default for new emails and replies, then click Save Changes

How to add to Mac Mail

  1. Select everything below this box (from the logo to the location)
  2. Copy it (Cmd+C)
  3. Open Mac Mail > Settings (Cmd+,) > Signatures tab
  4. Select your email account and click the + button to create a new signature
  5. Uncheck "Always match my default message font" at the bottom
  6. Click in the signature preview area on the right and paste (Cmd+V)
  7. Drag the new signature to your email account on the left to assign it
